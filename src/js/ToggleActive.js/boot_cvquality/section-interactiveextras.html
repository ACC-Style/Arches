4.3 Link Decorator

Links with the .link will auto decorate with an icon of the corresponding file type

.pdf Portable Document Format .doc Word Documents .txt Text Documents .ppt Power Points mailto: email links tel: Telephone fax: Fax _blank External Link icons will appear after any link that has target blank, or the href begins with http, or https

Example

A link that is a Pdf Document
A link that is a Word Document
An External link that is a Power Point Document
1-555-555-5555
test@abc.org
A link that is a Zip Document
 <a class="link" href="abc.pdf">A link that is a Pdf Document</a><br/>
 <a class="link" href="abc.doc">A link that is a Word Document</a><br/>

4.5 Lorem + Aspect Ratio

To create an image placeholder use the lorem-loader plus an aspect of the intended image on th parent container while waiting for the image.The placeholder will be a 100% of the container so size the container properly.

Examples

Default styling

 

.aspect_1x2

.aspect_1x1

.aspect_2x1

.aspect_3x2

.aspect_4x3

.aspect_16x9

.aspect_21x9
