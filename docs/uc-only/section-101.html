101 UC: Margin & Padding

Padding and Margins are the most common usage for Utility Classes. Spacing UC's are responsive and can be postfixed with -medium and with -large to have finer control on the spacing of objects.

101.2 Margin

Is Responsive: breakpoint modifiers
Margin classes are mobile first with postfix modifier for '-medium'and '-large'variations for responsive sizing.

.m_auto.small-6 - Margin 0 auto .m_0 - All margins set to 0 .m_1.m-t_0:large - All margins set to 1px .m_2.m-t_0:large - All margins set to 1/4 of global margin .m_3.m-t_0:large - All margins set to 1/2 of global margin .m_4.m-t_0:large - All margins set to global margin .m_5.m-t_0:large - All margins set to 2x of global margin .m-t_4 - Top margin set to global margin .m-r_4 - Right margin set to global margin .m-l_4 - Left margin set to global margin .m-b_4 - Bottom margin set to global margin .m-t_n4 - Top margin set to negative global margin .m-r_n4 - Right margin set to negative global margin .m-l_n4 - Left margin set to negative global margin .m-b_n4 - Bottom margin set to negative global margin
