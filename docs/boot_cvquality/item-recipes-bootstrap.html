Jumbotron Jumbotron are hero places that can have larger marketing text with a colored background. By default Jumbotron is dark text on a light grey background the modifiers add some predefined colored backgrounds using the brand colors. The backgrounds were generated here https://qrohlf.com/trianglify-generator/.

.orange-blue orange to blue background .orange-grey orange to grey background .blue-grey blue to grey background .jumbotron-fluid removes the border radius and space from the sides so it goes full width. .soft softens the edges of the triangles in the background .blur blurs the back to a smooth gradient

**/ .jumbotron:before { -webkit-background-size: cover; -moz-background-size: cover; -o-background-size: cover; display: block; z-index: -1; position: absolute; top: 0; right: 0; bottom: 0; left: 0; background: url("../img/cvquality/pattern-grey.png") no-repeat center center; background-size: cover; content: ""; } .jumbotron.orange-blue:before { background-image: url("../img/cvquality/pattern-orange-blue.png"); } .jumbotron.orange-grey:before { background-image: url("../img/cvquality/pattern-orange-grey.png"); } .jumbotron.blue-grey:before { background-image: url("../img/cvquality/pattern-blue-grey.png"); } .jumbotron.soft:before { -webkit-filter: blur(3px); -moz-filter: blur(3px); -o-filter: blur(3px); -ms-filter: blur(3px); top: -3px; right: -3px; bottom: -3px; left: -3px; filter: blur(3px); } .jumbotron.blur:before { -webkit-filter: blur(25px); -moz-filter: blur(25px); -o-filter: blur(25px); -ms-filter: blur(25px); top: -25px; right: -25px; bottom: -25px; left: -25px; filter: blur(25px); } .jumbotron .container { z-index: 3; width: auto; }
