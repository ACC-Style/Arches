16.4 Ordered Cards

Having cards order on page to help users find there way through steps can some times be very helpful.

Example

First it is bad

This is a card.

It has an easy to override visual style, and is appropriately subdued.

Then it is not right

This is a card.

It has an easy to override visual style, and is appropriately subdued.

At last it is good

This is a card.

It has an easy to override visual style, and is appropriately subdued.
