2.5.3 Table Unshaded

`

removes the shadding for rows and footer giving a cleaner table. This table type is great for small data sets and removes the 100% wide table that is default in all other tables.

Example

table header table header table header
row 1 cell A row 1 cell B row 1 cell C
row 2 cell A row 2 cell B row 2 cell C
row 3 cell A row 3 cell B row 3 cell C
footer cell 
<table class="table unshaded">
  <thead>
    <tr>
      <th>table header</th>
      <th>table header</th>
      <th>table header</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>row 1 cell A</td>
      <td>row 1 cell B</td>
      <td>row 1 cell C</td>
    </tr>
    <tr>
      <td>row 2 cell A</td>
      <td>row 2 cell B</td>
      <td>row 2 cell C</td>
    </tr>
    <tr>
      <td>row 3 cell A</td>
      <td>row 3 cell B</td>
      <td>row 3 cell C</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">footer cell</td>
    </tr>
  </tfoot>
</table>
