3.2 Link Decorator

Links with the .link will auto decorate with an icon of the corresponding file type

File Effect
.pdf Portable Document Format
.doc Word Document
.txt Text Document
.ppt Power Point
mailto: Email
tel: Telephone
fax: Fax
target=_blank External Link icons will appear after any link
